# Socapi

Socapi is a developer tool that helps users connect their Pinterest account to publish, schedule, and manage Pinterest Pins and boards.

The app uses the Pinterest API only after a user authorizes access through Pinterest OAuth.

## Features

- Connect a Pinterest account
- Read Pinterest account information
- Read boards and pins
- Create and manage boards
- Create and publish pins
